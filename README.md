![Robot Framework](https://img.shields.io/badge/Robot_Framework-Automation-blue)
![Selenium](https://img.shields.io/badge/Selenium-Web_Testing-green)

This repository contains my practice work while completing the courses *Robot Framework Test Automation – Level 1* and *Level 2* by Bryan Lamb.  
The project demonstrates automated UI testing using Robot Framework with SeleniumLibrary through a series of progressively more advanced examples, ending with an end-to-end test scenario.

## What is covered

- Robot Framework fundamentals and test structure
- Browser automation using SeleniumLibrary
- Web element locators (CSS and XPath)
- Page Object Model implementation
- Custom keywords and reusable test logic
- Variables, lists and dictionaries in Robot Framework
- Conditional logic (IF/ELSE, loops)
- Command-line test execution and reporting
- End-to-end automation test scenario

## Repository structure

Each folder represents a separate exercise or concept from the course, progressing from basic Robot Framework syntax to more advanced automation scenarios.
